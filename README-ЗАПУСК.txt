=====================================
Steam Inventory Parser - Инструкция по запуску
=====================================

ТРЕБОВАНИЯ:
- Windows 7/8/10/11
- Node.js версии 16 или новее (скачать с https://nodejs.org/)

БЫСТРЫЙ ЗАПУСК:
1. Дважды кликните на install.bat для установки зависимостей
2. Дважды кликните на start.bat для запуска приложения
3. Откройте браузер и перейдите на http://localhost:5000

ФАЙЛЫ:
- install.bat - установка всех зависимостей
- start.bat - запуск приложения в режиме разработки
- start-production.bat - запуск в продакшн режиме (быстрее)

ИСПОЛЬЗОВАНИЕ:
1. Откройте настройки и добавьте Steam API ключ (если нужен)
2. Добавьте Steam аккаунты через SteamID или ссылку на профиль
3. Нажмите "Обновить инвентарь" для загрузки кейсов
4. Просматривайте статистику и цены на главной странице

ВАЖНО:
- Приложение создает локальную базу данных в папке data/
- Все данные хранятся только на вашем компьютере
- Для остановки приложения нажмите Ctrl+C в окне командной строки

РЕШЕНИЕ ПРОБЛЕМ:
- Если не запускается - проверьте, что установлен Node.js
- Если ошибки установки - запустите install.bat от имени администратора
- Если порт 5000 занят - закройте другие приложения или перезагрузите компьютер
