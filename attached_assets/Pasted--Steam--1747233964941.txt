Необходимо разработать программу для парсинга инвентарей Steam аккаунтов с возможностью:
Добавления и удаления аккаунтов
Получения списка кейсов в инвентаре каждого аккаунта
Получения текущих цен на эти кейсы с Steam Marketplace
Расчёта общей стоимости кейсов на каждом аккаунте
Отображения суммарной стоимости всех кейсов на всех аккаунтах
Требования к реализации
1. Основные функции
Менеджмент аккаунтов:
Добавление новых аккаунтов (по SteamID или custom URL)
Удаление аккаунтов из списка
Сохранение списка аккаунтов между сеансами
Парсинг инвентаря:
Получение списка всех кейсов в инвентаре
Фильтрация только кейсов (исключение других предметов)
Получение актуальных цен с торговой площадки Steam
Кеширование данных для уменьшения количества запросов
Расчёт стоимости:
Индивидуальная стоимость кейсов для каждого аккаунта
Суммарная стоимость всех кейсов на всех аккаунтах
Конвертация в разные валюты (по курсу Steam)
2. Интерфейс
Главное меню:
Отображение общей суммы стоимости всех кейсов
Кнопка "Обновить все" для актуализации данных
Список добавленных аккаунтов с краткой информацией
Детали аккаунта:
Список кейсов с их количеством и текущей ценой
Общая стоимость кейсов на аккаунте
Дата последнего обновления
3. Безопасность и маскировка
Запросы к Steam:
Реализация человеко-подобных интервалов между запросами
Случайные задержки между 1.5 и 4 секундами
Ротация User-Agent
Ограничение количества запросов в минуту
Обработка ошибок:
Защита от бана IP
Повторные попытки при ошибках
Уведомление о приватных инвентарях
Технические детали
1. Архитектура
Язык программирования: Python
GUI: PyQt или Tkinter
Для запросов: aiohttp/requests с прокси поддержкой
Для парсинга: BeautifulSoup/lxml
Хранение данных: SQLite или JSON
2. Steam API
Использование официального Steam Web API (если есть ключ)
Альтернативно: парсинг страниц инвентаря
Для цен: запросы к marketplace
3. Алгоритм работы
Получить список всех предметов в инвентаре через Steam API или парсинг
Фильтровать предметы, оставляя только кейсы (по appid и известным названиям)
Для каждого кейса получить текущую цену с торговой площадки
Рассчитать общую стоимость
Обновить интерфейс
